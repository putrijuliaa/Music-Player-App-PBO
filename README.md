## Music Player App

Pada Project ini kami membuat Music Player App dengan nama Jafa Music Player App menggunakan python tkinter.

## Features

- Loadsongs/add song
- Memilih lagu
- play song
- next song/previous song
- volume
- shuffle song
- search song
- reset song
- remove song
- track information
- duration and slider
- scroll playlist
