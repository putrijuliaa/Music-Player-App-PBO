## Music Player App

Pada Project ini kami membuat Music Player App dengan nama Jafa Music Player App menggunakan python tkinter.

## Features

- ladsongs/add song
- memilih lagu
- play song
- next song/previous song
- volume
- shuffle song
- search song
- reset song
- remove song
- track information
- duration and slider
- scroll playlist
